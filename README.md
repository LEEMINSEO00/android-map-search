# android-map-search
## Step 1. 카카오 로컬 API
### 구현 기능 목록

1. 검색어 입력 시 검색 결과 15개 이상 표시
2. 검색 데이터
   - 카카오 로컬 API 사용
   - 카카오 API 사용 위한 앱 키 외부 노출 X
3. 이미 구현 되어 있는 사항
   - 입력한 검색어
     - X 버튼 눌러 삭제
   - 검색 결과 목록
     - 세로 스크롤
     - 하나의 항목 선택 가능
       - 선택된 항목 -> 검색어 저장 목록에 추가
   - 저장된 검색어 목록
     - 가로 스크롤
     - X 버튼 눌러 삭제
     - 앱 재실행 시 유지
